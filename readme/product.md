### cmd.product

| name        | return value  | description   |
|-------------|---------------|---------------|
| `product`   | `100`         | Returns the product of all given values. |

The following example returns the product 1 * 2 * 3 * 4 * 5:

```js
cmd.product.with(1, 2, 3, 4, 5);
// 120
```
