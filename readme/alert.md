## Alert

### `cmd.alert(val1, ...)`

| name     | all or each?  | accepts args?  | return value        |
|----------|---------------|----------------|---------------------|
| `alert`  | each          | no             | `[undefined, ...]`  |

#### Example

The following example displays two alerts in sequence.

```js
cmd.alert('Hello World!', 'Will Smith here.');
```
